🎓 Project Details

Project Title:
AI-Based Disease Prediction and Medicine Recommendation System

Domain:
Artificial Intelligence & Machine Learning

Technologies Used:
Python, Flask, Machine Learning (SVC), HTML, CSS, Bootstrap

🚀 Project Objective

The objective of this project is to develop an intelligent healthcare support system that predicts diseases based on user-reported symptoms and recommends suitable medicines, precautions, diet, and workouts. This system aims to assist users in making quick and informed health decisions using AI.
